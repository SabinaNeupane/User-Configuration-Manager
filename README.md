# User-Configuration-Manager
A simple User Configuration Manager built in Python that demonstrates how to manage user preferences using dictionaries, functions, and basic validation logic.

This project allows you to add, update, delete, and view user settings in a structured and readable way. It is designed to showcase core Python concepts such as dictionaries, tuples, conditionals, and string formatting.
